    Hexo搭建个人博客

    准备工作

    • Node安装

      1、一般安装在D盘，默认会选中Add to PATH，所以一直点击next即可

      2、检验是否安装：在相应文件的目录下，按下shift+鼠标右键，输入node -v来检查安装版本

    • GitHub注册
      1、具体注册请自行百度
      2、建一个仓库（Repository），具体操作：点击Start a project –>Repository name下输入用户名.github.io（例如，Southpost.github.io）–>Creating repository即可

    • Git安装
      1、点击next，看到Use Window‘s default console windown选项选择，然后继续next，建议将它装在D盘，然后点击Finish就安装成功了，有没有感觉很Easy

    Github SHH配置

    • 打开安装好的git，输入命令
      • 打开GitHub –> settings –> SSH and GPG keys –> Title随便写 –> 在个人下载git的目录下寻找.SSH文件夹 –> 找到id_rsa.pub，利用记事本的方式打开，并复制里面的全部内容 –> 回到GitHub，粘贴刚才的内容到Key中 –>点击Add SSH Key即可

    Hexo之主题使用

    选择相应的问题进行下载（zip），然后将其放到themes文件夹下就OK了

    主题自定义补充

    想要修改主题的一些个性化设置，比如，扫描二维码你可以到themes文件夹下，找到img文件夹进行修改；也可以到_config.yml(主题目录下的，不是博客根目录下的)文件夹下进行修改
