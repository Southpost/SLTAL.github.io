开发前准备

java 8安装及配置环境变量

  • jdk的安装与配置是Java学习的第一步，下面记录一下具体过程。首先根据自己系统下载对应版本。下载地址http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
  • 先点击 接受 Accept License Agreement
    根据自己的操作系统选择不同版本下载
    安装的时候最好选择自定义安装目录，后续会用到，安装路径要记得。我自己自定义安装在D盘自己新建一个Java文件夹里面
  • 环境变量的配置
    1、右键“此电脑”—>“属性”—>“高级系统设置”—>“高级”—>“环境变量”—>“新建”：

    2、新建环境变量JAVA_HOME
    变量名：JAVA_HOME；
    变量值：自己的jdk安装目录(下图是举例)。

    3、在系统变量中找到“Path”，在“Path”里面点击“编辑”

    4、选择“新建”，新建两条路径
    %JAVA_HOME%\bin
    %JAVA_HOME%\jre\bin

    5、查看是否成功
